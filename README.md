# SilentFlow

SilentFlow is a modern, offline-first study tracker designed to help students manage tasks, maintain focus, and track personal growth through a simple and intuitive interface.

The application combines task management, focus sessions, and daily reflections into a single workspace, allowing users to build consistent study habits and monitor their progress over time.

## Overview

SilentFlow was created with a simple idea:

Progress is often invisible in the short term. By tracking effort consistently, students can gain a clearer understanding of their growth and maintain long-term discipline.

## Features

### Task Management
- Create and manage study tasks
- Mark tasks as completed
- Delete completed or unnecessary tasks
- Track remaining work

### Focus Timer
- Dedicated study timer
- Start, pause, and reset functionality
- Designed to encourage focused work sessions

### Daily Reflections
- Record learning outcomes and achievements
- Automatically saved locally
- Build a personal learning history

### Progress Dashboard
- Overview of study activity
- Task completion tracking
- Productivity monitoring

### Offline Support
- Works without an internet connection
- Data stored locally in the browser
- Fast and lightweight experience

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/SilentFlow.git
```

Navigate to the project directory:

```bash
cd SilentFlow
```

Open `index.html` in your browser.

No additional dependencies or installation steps are required.
